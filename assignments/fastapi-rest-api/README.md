# 📘 Assignment: FastAPI REST APIs

## 🎯 Objective

Learn how to build a RESTful API using the FastAPI framework by creating endpoints, defining data models, and handling requests and responses.

## 📝 Tasks

### 🛠️ Define API Models and Routes

#### Description
Create a FastAPI application with data models for a simple resource and implement basic routes for creating and retrieving items.

#### Requirements
Completed program should:

- Use FastAPI to define an application instance
- Create a Pydantic model for the API resource
- Implement a route to add a new item with `POST`
- Implement a route to list all items with `GET`

### 🛠️ Implement CRUD Operations

#### Description
Add update and delete support to the API so clients can modify and remove resources.

#### Requirements
Completed program should:

- Implement a route to retrieve a single item by ID
- Implement a route to update an item using `PUT` or `PATCH`
- Implement a route to delete an item using `DELETE`
- Use an in-memory store such as a list or dictionary for item data

### 🛠️ Add Validation and API Documentation

#### Description
Add request validation and verify the API documentation generated by FastAPI.

#### Requirements
Completed program should:

- Validate request payloads using Pydantic model fields and types
- Handle invalid input with clear error responses
- Confirm the API documentation is available at `/docs` or `/redoc`
- Start the app with Uvicorn and test endpoints manually or with a simple HTTP client
