# 📘 Assignment: Statistics with Python

## 🎯 Objective

Practice data analysis and statistical calculations using Python with `pandas` and `numpy` by exploring a dataset and summarizing its key metrics.

## 📝 Tasks

### 🛠️ Load and Explore Data

#### Description
Load the provided dataset with pandas and explore its structure, columns, and basic summary statistics.

#### Requirements
Completed program should:

- Read `data.csv` into a pandas DataFrame
- Print the first 5 rows of the dataset
- Display the dataset shape and column names
- Show summary statistics for numeric columns

### 🛠️ Compute Statistical Measures

#### Description
Use `numpy` and pandas to calculate core statistical metrics and compare results.

#### Requirements
Completed program should:

- Compute mean, median, variance, and standard deviation for at least two numeric columns
- Use `numpy` functions for one or more statistics
- Print results in a clear, labeled format

### 🛠️ Analyze Relationships and Results

#### Description
Analyze relationships in the data and present findings from the computed statistics.

#### Requirements
Completed program should:

- Calculate a correlation between two numeric columns
- Identify a column with the highest variance or spread
- Print at least one short conclusion based on the dataset
